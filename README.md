# 💡 Fluxo de Ideias (IdeaFlow)

O **Fluxo de Ideias** é um caderno de notas digital minimalista e eficiente, projetado para capturar pensamentos rapidamente. O projeto foi desenvolvido com foco em performance, persistência de dados local e uma experiência de usuário limpa, adaptável tanto para desktop quanto para dispositivos móveis.

## 🚀 Funcionalidades

- **CRUD de Notas:** Crie, visualize, edite e exclua notas facilmente.
- **Persistência Local:** Suas notas ficam salvas no navegador (LocalStorage), sem necessidade de banco de dados externo.
- **Barra de Busca:** Filtre suas ideias em tempo real por título ou conteúdo.
- **Modo Escuro (Dark Mode):** Alternância visual para maior conforto em ambientes com pouca luz.
- **Interface Responsiva:** Otimizado para uso no celular como um Web App (PWA).
- **Customização:** Ajuste de tamanho de fonte para melhor leitura.

## 🛠️ Tecnologias Utilizadas

- [Next.js](https://nextjs.org/) - Framework React para produção.
- [Tailwind CSS](https://tailwindcss.com/) - Estilização moderna e rápida.
- [Lucide React](https://lucide.dev/) - Biblioteca de ícones minimalistas.
- [LocalStorage API](https://developer.mozilla.org/pt-BR/docs/Web/API/Window/localStorage) - Armazenamento de dados no cliente.
